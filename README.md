### Lista Numerada:
1. Uma vez asterisco ou underline = itálico
2. Duas vezes asterisco ou anderline = negrito
3. Um asterisco e um underline juntos = itálico + negrito
4. Três asteriscos = uma linha
5. Duas vezes til = palavra riscada
6. Titulos
   1. Um hashtag = título grande
   2. Dois hashtags = título médio
   3. Três hashtag = título pequeno;
7. Emojis = Dois pontos e a palavra em inglês. Ex.: 🖖 🤙 🫶
